Software I Use

Priorities

Web Browser

Even though the CEO and company have made some seriously questionable descisions during the lifetime of this web browser my current choice is Brave which you download with my link here. I've used Vivaldi, Chomium and Firefox in the past on Linux and they're all about the same but Brave comes with internet money so it's a plus for me.

Terminal

I've cycled through a bunch of different terminals like URXVT, Kitty, and st but the one that I've currently settled on is Alacritty. It does everything that I need from a terminal, easy key remapping, true color, it's lightning fast and is dead simple to configure. Unlike certain other terminals it's not just a walking LOC meme.

Video Player

Why even bother asking about my video player, if you're not using MPV you're doing it wrong. The only other acceptable answer is MPlayer which is just MPV without the GUI controls.

Image Viewer

I need to view images not file my taxes, just give me some simple window that shows me the image I want to look at and Sxiv seems to do that job as well as I could possibly expect. Feh is also a really good alternative as well and doubles as a wallpaper setter but I have my reasons for using XWallpaper for that particular job.

Crypto Wallet

I have secondary wallets for things like Tron staking but my primary wallet for the time being is Exodus, it has all the coins supported that I could possibly want and also comes with a pretty slick interface, I'm not really sure what else I would move to if I stopped using this one.

Text Editor

Right now it's obviously going to be Vim, or I guess it's actually NeoVim but for the most part the behave the same way it's not until you start spending way too much time customizing your editor that you really start to spot the difference.

Video Editor

Even though I hate it and I want to use something better for now I'm kind of stuck using Kdenlive, I have tried out other editors like OpenShot which is seriously lacking in features and Olive which has a serious memory leak so for now I'll stay where I am. When Olive is stable I'm without a doubt switching over to that instead.

Image Editor

Most of my image editing work is done with thumbnails in mind, I don't typically find myself doing much else and I find that GIMP seems to do basically all if not everything I need from an image editor. If you've come from using Photoshop, GIMP may seem a little bit clunky and in many ways that's true but it's capable of most of what normal people use Photoshop for.

Video/Audio Recording

For recording all of my videos I use OBS because having switchable scenes is really nice and much easier to handle than just doing it in post when I need to frequently move around the camera. As for audio recording typically this is done inside of OBS alongside the video but when I need to do it seperately my go to choice is Audacity.

Email Client

Thunderbird simply just Thunderbird, you can make the argument that you just need a terminal based email client but I do like being able to see images in the emails I receive. Thunderbird also comes with some other nice things like a calender, a schedule and various other ueful things that I'd be too lazy too check if they were in another app.

Operating System

It should be pretty obvious that I'm running Linux but what distro am I running well obviously I'm running Arch btw. I'm considering trying out Artix on my next install but for now I'm happy on Arch.

Fonts

For my web browser I like having my fonts look like they did back when I was on Windows, I know that a lot of people like having custom fonts in their browser but that's not my thing so I always install the Noto Fonts package. On my desktop, in my terminal and everywhere else on my system I'm currently running JetBrains Mono.

Window Manager

I've been considering trying out other options like DWM, Qtile, Berry and a few others and back when I first started out on Linux I was running i3 which was great but then I discovered BSPWM and ever since that day I've just been hacking on this window manager to make it behave exactly the way I want.

File Manager

I've got 2 file managers installed on my system at all times, a GUI and a terminal file manager. Most of the time I use the terminal file manager but when I'm interacting with GUI applications the GUI file manager comes out but any I'm currently running lf as the terminal choice and PCManFM as the GUI choice.

Torrent Client

Currently I'm just using Transmission, and to be completely honest there's nothing keeping me attached to it, I'm pretty much only using it because it's what I've been since I first installed Linux and I haven't been bothered to try out anything else. As for the interface I'm just using tremc which is a python 3 rewrite of transmission-remote-cli.

Office Suite

Are there even any choices of Linux, on the rare chance I need an office suite I'll be running Libre Office, typically I only use it for dealing with my taxes and documents that other people from Uni send me, if I have control I wouldn't be using it.

RSS Reader

Typically I only use RSS for looking at various Reddit feeds because I hate the site but it's great for farming content so my RSS Reader of choice is currently Newsboat, it does everything I need it to do and runs in my terminal so it's an obvious choice.

Hardware I Use

Monitor

I bought an ASUS VL279HE, it's a completely usable monitor but it does have some annoying problems like the stand which is really wobbly and I already have a dead pixel on it which isn't great but it's ok on a budget.

CPU

Originally I was supposed to have AMD Ryzen 9 3900X in this system but due to issues during the build process I had to replace it and the only thing I could get was an AMD Ryzen 5 3600X, which to be completely honest is fine.

GPU

GPU rendering on Linux is sort of a meme so I really on got the Gigabyte Radeon RX 570 so that I'd have some sort of video out and it'll be fine for streaming some light games, if I get into streaming PC games I'll replace it with something bigger in the AMD lineup. Not due to being a fan of AMD, it's just that Nvidia is hell on Linux.

Motherboard

The Asus PRIME X570-P is a motherboard, I don't know what else to say about it, it's not the cheapest X570 board and it's not the most expensive and it has almost too many USB ports which is nice. I like it.

Powersupply

Is the EVGA SuperNOVA G+ 650W way overkill for my system, yeah without a doubt, I should have dropped down to the 550W model instead but this will do for now and it gives more room to grow than I ever plan to do.

Case

Cheap cases have come along way since the last time I built a PC and the InWin 101 is one of the best examples of this, it's not best designed case interally but it does the job and doesn't slice your hand open. It doesn't come with any fans which is sort of a bumber but it does have a full glass side panel which is pretty neat.

Cooling

For my CPU cooler I went with the Noctua NH-U12S which being a Noctua CPU cooler is an absolute beast and is more than I need for my CPU. For the case fans I went with a 3 pack of Corsair Air Series AF120 with red LEDs to go with the AMD CPU and GPU, not really they were just the only ones in stock so they were what I bought.

Memory

I forgot to order RAM with the rest of my parts and the Corsair Vengance LPX 32GB was the last 32GB kit left in stock at my local computer store, I didn't need low profile memory in this system but there's no harm in having it.

Storage

I'm rocking 3 drives in my desktop an Corsair Force MP510 480GB as my boot and root drive, a Seagate 2TB BarraCuda as my home drive and a Crucial BX500 120GB 3D which currently exists as a Windows 10 drive but will probably just turn into the drive I nuke when I want to run an OS on hardware rather than in a VM.

Peripherals

The keyboard I'm using which I've showed from time to time in the podcast is a Cougar Vantar, which is a scissor switch keyboard (not to be confused with butterfly switch) because scissor switches are hands down a better typing experince than the alternatives, when I replace it I'll probably get something like the Cherry KC 6000 Slim. For my mouse I couldn't tell you what it is, it's just some random no name mouse that does the job and that's about all.

Audio Output

In the podcast you'll see me wearing earphones, these are a pair of Logitch CX 3.00, they're not super cheap but they're not super expensive, I've been buying these for years now and they sound pretty good. When I just want to chill out and use my speakers my go to is an LG 300W Soundbar it's not the best sounding thing out there but it's good enough for watching YouTube and anime.

Microphone

I'm currently rocking a Blue Yeti it's pretty much the standard YouTube mic but I wouldn't reccomend it since it has some fundamental hardware flaws. As for my microphone arm it's just some cheap thing I got off of Ebay and the same is true for the wind foam, you don't need to spend much on this stuff. My shock mount is Knox Gear Mount but you can find the same one rebranded as Auphonix and presumably other names as well.

Webcam

If all you need is a view of your face the Logitech C920 Pro is good enough and if you're streaming games, most of the time the camera will so small that a better camera won't make much of a difference.

Lighting

Lighting is always important but especially so when using a webcam, I currently use 2 Neewer 176 point lights to light up my face one to the left of the scene and one to the right, they're both on cheap stands I got off ebay. My background is lit up by some random lamp I got from Bunnings that I decided to stick a stupid bright bulb into and it seems to do the job.

Laptop

It's basically been retired at this point but my laptop is an Acer Swift 3 which was the display unit so I got a bit of a discount on it, it was a good laptop back when I bought it but it's got a lot of hardware problems now and it's sort of at the end of it's life.
